# Многолетник — Сайт саженцев многолетних цветов

Одностраничный сайт (landing page) для продажи саженцев многолетних цветов с дачи.

## Содержание сайта

- **Шапка (Header)** — фиксированное меню с логотипом, навигацией и адаптивным бургер-меню
- **Главный экран (Hero)** — крупный заголовок, описание, кнопки призыва к действию, статистика (50+ сортов, 7 лет опыта, 98% приживаемость)
- **О нас (About)** — информация о преимуществах с сеткой ценностей (натуральность, адаптация, разнообразие, гарантия качества)
- **Каталог (Catalog)** — карточки растений с изображениями, ценами и кнопками заказа (пионы, ирисы, примулы, флоксы, клематисы, георгины)
- **Преимущества (Features)** — 4 причины выбрать многолетники
- **Что вы получаете (Highlight)** — блок с описанием услуг и изображением
- **Отзывы (Testimonials)** — карточки с отзывами клиентов
- **Призыв к действию (CTA)** — контактный телефон
- **Подвал (Footer)** — ссылки на категории, информацию для покупателей, контакты

## Технические особенности

- Чистый HTML + CSS + JS (без фреймворков и библиотек)
- Адаптивная вёрстка (десктоп, планшет, мобильные устройства)
- CSS-анимации: появление элементов при скролле (Intersection Observer), свечение логотипа, пульсация
- Кастомные CSS-свойства (custom properties) для цветовой схемы
- Плавные переходы и hover-эффекты
- Фиксированная шапка с эффектом тени при скролле
- Мобильное бургер-меню

## Общие выводы

1. Сайт выполнен как одностраничный лендинг с современным дизайном в природной цветовой гамме (зелёные, кремовые, терракотовые оттенки).
2. Реализована полная адаптивность под все устройства — от широких экранов до мобильных телефонов.
3. Интерфейс интуитивно понятен: посетитель может ознакомиться с ассортиментом, узнать о преимуществах, прочитать отзывы и связаться с продавцом.
4. Использование CSS-анимаций и плавных переходов создаёт приятный пользовательский опыт.
5. Кодовая база легковесна — один HTML-файл, не требуется сборка или установка зависимостей.
6. Проект готов к размещению на любом хостинге или GitHub Pages.
