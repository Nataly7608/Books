# Задача: Переименование лотов книг для продажи

## Цель
Переименовать файлы фотографий книг на диске `D:\` для лотов продажи.  
Новый формат имени: `<номер_лота>_<ФИО автора>_<Название книги>.<расширение>`

## Этапы

### 1. Установка и настройка OCR
- Tesseract OCR v5.4 установлен
- Язык `rus` (русский) загружен

### 2. Распознавание текста (OCR)
Команда для одного файла:
```powershell
& "C:\Program Files\Tesseract-OCR\tesseract.exe" "<файл>" stdout --tessdata-dir "$env:LOCALAPPDATA\Tesseract-OCR\tessdata" -l rus --psm 6
```

### 3. Формирование таблицы соответствия имён
Файл: `rename_table.md`

### 4. Проверка пользователем
Пользователь проверяет и исправляет ФИО авторов и названия книг.

### 5. Переименование
Команда переименования:
```powershell
Rename-Item -LiteralPath "<старый путь>" -NewName "<новое имя>"
```

## Примечания
- OCR-распознавание обложек книг даёт низкое качество из-за декоративных шрифтов и сложного фона
- Рекомендуется ручная проверка всех распознанных данных
- Файлы не удаляются и не перемещаются между папками
