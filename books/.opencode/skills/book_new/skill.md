# Skill: Переименование лотов книг (book_new)

## Описание
Автоматическое переименование файлов фотографий книг на съёмном диске `D:\` для лотов продажи. Новое имя файла содержит номер лота, ФИО автора и название книги.

## Формат нового имени
```
<номер_лота>_<ФИО автора>_<Название книги>.<расширение>
```

Пример: `_DSC0080_1_Иванов И.И. Название книги.JPG`

## Порядок работы

### 1. OCR-распознавание
Запускается Tesseract OCR (v5.4) с русским языком для каждой фотографии лота. Для каждого лота обрабатываются все его фото (обложка, содержание и т.д.), выбирается лучший результат.

Команда:
```powershell
& "C:\Program Files\Tesseract-OCR\tesseract.exe" "<путь к фото>" stdout --tessdata-dir "$env:LOCALAPPDATA\Tesseract-OCR\tessdata" -l rus --psm 6
```

### 2. Извлечение ФИО и названия
Из OCR-текста извлекаются строки, содержащие:
- ФИО автора (в формате "И. О. Фамилия" или "Фамилия, И. О.")
- Название книги (в кавычках «» или после ФИО)

### 3. Формирование таблицы
Результаты сохраняются в MD-файл с таблицей формата:
| Папка | Старое имя | Новое имя | Статус |
|-------|-----------|-----------|--------|

### 4. Переименование
После проверки пользователем команда переименования:
```powershell
Rename-Item -LiteralPath "<старый путь>" -NewName "<новое имя>"
```

## Директории обработки
- `D:\Книги` — основная папка книг (с подпапками по сериям)
- `D:\Книги 3-лоты Наташи` — дополнительная папка

## Зависимости
- Tesseract OCR 5.4+ (установлен)
- Русский язык: `rus.traineddata` (загружен)
- PowerShell 7+
