# Skill: Переименование лотов книг (book_new)

## Описание
Автоматическое переименование файлов фотографий книг на диске `D:\` для лотов продажи. Новое имя содержит исходное имя файла, номер лота, ФИО автора и название книги.

## Формат нового имени
```
<исходное_имя>_<номер>_<ФИО автора>_<Название книги>.<расширение>
```

Пример: `_DSC4676_1_В.Каверин Вл.Носов_Новое зрение.JPG`

## Порядок работы

### 1. Составление таблицы
- Собрать список всех файлов в целевой папке
- Создать MD-таблицу с колонками: №, Исходный файл, ФИО автора, Название книги, Новое имя
- Пользователь заполняет ФИО автора и название книги

### 2. OCR-распознавание (если нужно)
Запускается Tesseract OCR (v5.4) с русским языком для каждой фотографии лота.

Команда:
```powershell
& "C:\Program Files\Tesseract-OCR\tesseract.exe" "<путь к фото>" stdout --tessdata-dir "$env:LOCALAPPDATA\Tesseract-OCR\tessdata" -l rus --psm 6
```

### 3. Переименование
Скриптом Python или PowerShell:
```powershell
python books/rename_pisatelei.py
```

## Директории обработки
- `D:\Книги` — основная папка книг (с подпапками по сериям)
- `D:\Книги 3-лоты Наташи` — дополнительная папка

## Зависимости
- Tesseract OCR 5.4+ (установлен)
- Русский язык: `rus.traineddata` (загружен)
- PowerShell 7+
- Python 3+
- (опционально) Ollama + llava:7b для альтернативного OCR
